4 MAIN CATEGORIES OF SQL STATEMENS
Data definition language (DDL)
Data management language (DML)
Data control language (DCL)
Data query language (DQL)

Data definition language (DDL)
CREATE : USED TO CREATE NEW DATA BASE OBJECTS,,SUCH AS TABLES, INDEXES, NAD VIEWS.
ALTER : USED TO MODIFY THE STRUCTURE OF EXISTING DATABASE OBJECTS.

Data Manipulation language (DML)
INSERT ; ADDS NEW RECORDS TO A TABLE 
UPDATE : MODIFIES EXISTING RECORDS IN A TABLE.

Data control language (DCL)
GRANT : PROVIDES SPECIFIC PRIVILEGES TO USER OR ROLES.
REVOKE: REMOVES SPECIFIC PRIVILEGES FROM  USER OR ROLES.

Data query language (DQL)
SELECT : RETRIEVES DATA FROM ONE OR MORE TABLES BASED ON SPECIFIED CRITERIA.

SEQUENCE OF EXECUTION  ( FIRST WE GET HUNGRY SO ORDER LUNCH)
RUNS TO SQL STEP 

STEP 	CLAUSE 	PURPOSE
1	FROM	SPECFIES THE DATA SOURC ( TABLES,VIEWS,JOINGS)
2	WHERE	ROW BASED ON SPECIFIED CONDITION
3	GROUPBY 	GROUPS ROWS INTO SUBSET ON ONE OR MORE COLUMNS FOR AGGREGATION
4	HAVING	FILTER GROUP CREATED BY GROUPBY BASED ON CONDITION
5	SELECT	SPECIFIC COLUMNS , CALCULATED EXPRESSION, OR AGGREGATED VALUES FOR THE OUTPUT.
6	ORDERBY	SPECIFIED COLUMNS IN ASCENDING(ASC)OR DESCENDING (DESC) ORDER.
7	LIMIT	RESTRICTS THE NUMBER OF ROWS – RETURNED IN THE RESULT SET.
